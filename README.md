# graduation_test-trump_randomselector
web研の課題。トランプをランダムに選ぶ

## 機能
- 4*13に並んだカードの中から1枚を選ぶことができる。選んだ時、そのカードが画面の中央に表示される。
- カードの中身は引くたびに毎回、シャッフルされる。

## 用途
- トランプを個々で確認したいとき。(他人と被ることがある)
- ひまつぶし

## 使用技術
- typescript
- tilewindcss
- next.js
