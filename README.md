# graduation_test-trump_randomselector
web研の課題。神経衰弱。

## 機能
- トランプを2枚引き、2組のペアを作ってクリアするまでの手数を記録する。
- トランプをクリックしたとき、または2組のペアが揃わず戻るときに回るアニメーションがついている。
- リスタートボタンで再挑戦できる。

## 用途
- ゲームである。
- 神経衰弱ができる。
- 1つのデバイス内で他人との対戦はできないが、
複数のデバイス同士で手数の多さを競うことができる。

## 使用技術
- typescript
- tilewindcss
- next.js
